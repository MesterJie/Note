# 快捷键

## 快捷键设置

```txt
打开文件->首选项->键盘快捷键
```



```txt
Alt+ ↑ / ↓	向上/向下移动行 Move line up/down
Shift+Alt + ↓ / ↑	向上/向下复制行 Copy line up/down
Ctrl+Shift+K	删除行 Delete line
Ctrl+Enter	在下面插入行 Insert line below
Ctrl+Shift+Enter	在上面插入行 Insert line above
Ctrl+Shift+\	跳到匹配的括号 Jump to matching bracket

Alt+PgUp / PgDown	向上/向下滚动页面 Scroll page up/down
Ctrl+K Ctrl+[	折叠（未折叠）所有子区域 Fold (collapse) all subregions
Ctrl+K Ctrl+]	展开（未折叠）所有子区域 Unfold (uncollapse) all subregions
Ctrl+K Ctrl+0	折叠（折叠）所有区域 Fold (collapse) all regions
Ctrl+K Ctrl+J	展开（未折叠）所有区域 Unfold (uncollapse) all regions
Ctrl+K Ctrl+C	添加行注释 Add line comment
Ctrl+K Ctrl+U	删除行注释 Remove line comment
Ctrl+/	切换行注释 Toggle line comment
Shift+Alt+A	切换块注释 Toggle block comment

移动到定义处:   F12
查看定义处缩略图(只看一眼而不跳转过去):    Alt+F12

删除光标右侧的所有内容(当前行):   Ctrl+Delete
扩展/缩小选取范围： Shift+Alt+Right 和 Shift+Alt+Left

=============自己觉得特别有效的=========================
Ctrl+Shift+[ 折叠该段代码
Ctrl+Shift+] 打开该段代码
Ctrl+] / [	左缩进/右缩进
Ctrl+End：移动到文件结尾
Ctrl+Home：移动到文件开头 
撤销上一步操作: Ctrl+Z
手动保存:   Ctrl+S
Home:当前行首
END:当前行尾
选择从光标到行尾的内容:   Shift+End
选择从光标到行首的内容： Shift+Home
多行编辑(列编辑):   Alt+Shift+鼠标左键 或 Ctrl+Alt+Down/Up
ctrl+ 左键或者右键 ：移动当前文件到另一半窗口
Caps +Alt 上下方向键 -----移动某一行代码
向上或向下复制一行:   Shift+Alt+Up 或 Shift+Alt+Down
不选中在某一行Ctrl+C-----复制整行
不选中在某一行Ctrl+x-----剪切整行
ALT +方向键----切换工作文档
CTRL +左右方向键移动光标到单词首和尾
CTRL + \ 窗口分区-------CTRL +F4关闭
shift +ALT +F 自动整理代码格式
CTRL +F查找之后通过Enter键向下查找，通过Shift +Enter 向上查找，Ctrl+H替换操作
在当前行下方插入一行:   Ctrl+Enter
在当前行上方插入一行:   Ctrl+Shift+Enter
Ctrl+p搜索文件
Alt+数字键---切换窗口（注意是英文上面对应的数字键）
Ctrl+数字键---切换和新建编辑窗口
Ctrl+B 关闭打开左侧窗口
Ctrl+D选中光标所在的那个单词
shift + alt + ⬇/⬆ 复制光标所在行，并创建下一行，把内容粘贴到下一行
Ctrl + K + w 关闭所有编辑文档
Ctrl+Shift+F全局搜索--->Alt+Enter将搜索结果打印
Ctrl+左方向键，移动到选中单词的最前面，+右方向键移动到最后面
Ctrl+Shift+\,在相对应的花括号之间跳转
只需要在光标移动的命令基础上再按下Shift键即可，此时光标会移动，文本也会选择。例如：home和end加上Shift,行选中。Ctrl+Home+Shift.......

```
